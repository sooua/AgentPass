# AgentPass logo — delivery package

The mark is a **keyhole plate**: a rounded square holding a keyhole (credential +
access). Three background-ready variants, all built on the same geometry and the
brand palette. Scale uniformly; keep clear space ≥ half the plate height.

| File | Plate | Keyhole | Use on |
|------|-------|---------|--------|
| `agentpass-terracotta.svg` | Terracotta `#c96442` | Ivory `#faf9f5` | Brand moments, hero, splash |
| `agentpass-light.svg` | Near-black `#141413` | Terracotta `#c96442` | Light backgrounds |
| `agentpass-dark.svg` | Sand `#e8e6dc` | Terracotta `#c96442` | Dark backgrounds |

For the mark + wordmark side by side, use `../lockup-horizontal.svg`. For OS app
icons use `../app-icon.svg`. Full brand guidance: [`../ICON.md`](../ICON.md).
